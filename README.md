# Privacy-Sofia

Página de captura estilo Privacy para Sofia.

## Estrutura

```
privacy-sofia/
├── index.html         # Página principal
├── css/
│   ├── index.css      # Estilos principais
│   ├── popup.css      # Estilos do popup
│   └── all.min.css    # Font Awesome
├── js/
│   ├── main.js
│   ├── button.js
│   └── popup.js
├── images/
│   ├── capa.jpg       # Imagem de capa
│   ├── perfil.jpg     # Foto de perfil
│   ├── thumb1.jpg
│   ├── thumb2.jpg
│   └── foto1.jpg
├── media/
│   ├── video1.mp4
│   └── video2.mp4
└── fonts/

## Como usar

1. Adicione as imagens na pasta `images/`
2. Adicione os vídeos na pasta `media/`
3. Configure os links de pagamento no `index.html`
4. Faça o deploy
